This project presents an exploratory data analysis of a US insurance dataset using Python in a Jupyter Notebook environment. The focus of the analysis is to derive meaningful insights from structured data without relying heavily on visualizations.

Key aspects of the project include:

Data cleaning and preprocessing
Statistical analysis of insurance charges
Exploration of relationships between variables such as age, BMI, smoking status, and region
Identification of key factors influencing insurance costs
Use of Python libraries such as Pandas and NumPy for efficient data handling

The goal of this project is to demonstrate strong analytical thinking and the ability to extract insights from raw data through code-based analysis rather than visual representation.
